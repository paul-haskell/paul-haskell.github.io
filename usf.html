Stories

Paul has been teaching Computer Science at the University of San Francisco since 2022. He teaches several introductory courses that cover:

  • Java programming
  • Object oriented programming, inheritance, and polymorphism,
  • Advanced data structures such as linked lists, rings and trees,
  • Algorithm complexity,
  • Recursion,
  • Basic graphics.

A highlight of Paul's courses are the projects in which students write software to play multiplayer games against other students' programs. Past games have included Blackjack, Acey-Deucey, Poker, and Liar's Dice. At the end of each project, students compete for prizes in a Game Night to see who's programs have the best strategy (or most luck).
